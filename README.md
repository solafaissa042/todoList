📝 MyTodosPro – Advanced React To-Do App

🚀 Project Overview
MyTodosPro is a task management web application built with React using modern state management patterns including useReducer and Context API.
The project demonstrates scalable state architecture beyond basic useState, making it closer to real-world React applications.

✨ Features
Add new tasks
Mark tasks as completed
Delete tasks
Global state management using Context API
State logic handled with useReducer
Persistent storage using Local Storage
Dynamic and responsive UI

🛠 Tech Stack
React (Functional Components)
React Hooks (useReducer, useContext, useEffect)
JavaScript (ES6+)
HTML & CSS
Local Storage API

🧠 Architecture & Concepts
This project demonstrates:
Centralized state management with useReducer
Global state sharing using React Context API
Separation of concerns between UI and state logic
Controlled components for form handling
Side-effects handling with useEffect
Clean and scalable component structure

💡 Why This Project Matters
Instead of managing state locally with multiple useState hooks, this project implements a more scalable pattern using Reducer + Context, similar to how larger React applications manage global state.
